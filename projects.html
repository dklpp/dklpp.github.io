BERT-based models fine-tuning for Lithuanian language
Python, PyTorch, BERT models, NLTK
Finetuned the base RoBERTa model for Lithuanian language. Based on this, trained and fine-tuned a BERT model for Named Entity Recognition (NER) task to distinguish 8 different entities. Achieved outstanding token accuracy of 99.2% on the test data. Evaluated the model with broad metrics like token precision, token recall, token F1-score, token entity exact precision, token entity exact recall, Matthews Coefficient, where each metric achieved more than 91% of accuracy.
Malnutrition Risk Prediction
Python, Machine Learning, Scikit-Learn, Exploratory Data Analysis, Statistical methods
Conducted Exploratory Data Analysis (EDA) on a relatively high dataset from Kitchener-Waterloo hospital with 7845 observations and 1385 features. Engineered a missing values imputation strategy using Generative Neural Networks and standard ML algorithms, improving the accuracy from 60% to 92% with imputation strategy. Performed advanced feature selection using complex feature importance estimation pipeline, which identified the most impactful features, reducing the dimensionality of features by 97% and preserving the most important information.
