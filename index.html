Projects

  • Abstractive and Extractive Text Summarization for Lithuanian and English languages
    Python · PyTorch · NLTK · NLP · LLM · LoRA/Q-LoRA fine-tuning · MLflow · Optuna
    Engineered a pipeline for abstractive and extractive summarization models for Lithuanian language. Fine-tuned various classic and LLM approaches, which included models mT5, mBART, BertSum, and LLM models Gemma, Llama 3.1, Mistral, Qwen. Managed to increase the Rouge-2 F1 score from initial scores of 10.1% up to 40.5% which depicts a high bigram matching, and also achieved BertScore semantic similarity of 92%. Used different fine-tuning optimization techniques.
  • BERT-based models fine-tuning for Lithuanian language
    Python · PyTorch · NLTK · NLP · NER
    Fine-tuned RoBERTa for Named Entity Recognition, achieving 99.2% token accuracy across 8 entity classes.
  • Malnutrition risk prediction
    Python · Scikit-Learn · GNN imputation · feature selection
    Analyzed clinical data with imputation strategies, improving accuracy from 60% to 92% and reducing features by 97%.
  • Parkinson’s disease automated detection
    Signal processing · PyTorch · time-series · classification
